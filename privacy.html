Política de Privacidad

Última actualización: abril 2025

Gracias por usar la extensión Exportar Chat WhatsApp. Esta política de privacidad describe cómo manejamos tu información.

1. Información que recopilamos

La extensión no recopila, almacena ni transmite ningún dato personal o sensible. Todos los datos procesados (como el texto visible de tus chats) se manejan exclusivamente dentro de tu navegador y no se envían a ningún servidor.

2. Uso de permisos

La extensión solicita permisos mínimos para poder funcionar:

  • scripting: para ejecutar funciones dentro de WhatsApp Web cuando el usuario lo solicita.
  • activeTab: para acceder temporalmente a la pestaña activa al presionar el botón de la extensión.
  • host_permissions: para acceder únicamente a web.whatsapp.com.

Estos permisos no se utilizan para monitoreo, rastreo ni recolección de datos.

3. Compartición de datos

No compartimos ni vendemos ningún tipo de datos. La extensión funciona de manera 100% local.

4. Cambios en esta política

Esta política puede actualizarse si se agregan nuevas funcionalidades. Te recomendamos revisarla ocasionalmente.
